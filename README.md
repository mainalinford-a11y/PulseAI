# JobMatch AI

AI-powered job matching platform that analyzes your CV against job postings and sends personalized match reports.

## Features
- Daily automated job searches
- AI-powered CV matching (0-100% score)
- Detailed qualification analysis
- Email reports with top matches
- Multi-tier subscription model

## Tech Stack
- Frontend: Next.js 14, TypeScript, Tailwind CSS
- Backend: Next.js API Routes
- Database: PostgreSQL
- Automation: n8n
- AI: Ollama (DeepSeek)
- Deployment: Vercel
